Mindful Muse is an AI-powered companion designed to support users with mindfulness, emotional awareness, and positive reflection. This project was developed as part of an academic assignment using AI-assisted development tools, as instructed by faculty.

## Project Overview
Mindful Muse acts as a digital companion that interacts with users in a calm and supportive manner. The goal of the project is to explore how AI can be used responsibly to promote mental well-being and self-reflection.

## Role of AI in This Project
AI tools were used to:
- Generate the initial project structure and UI
- Assist in writing and organizing code
- Support conversational flow design
- Improve styling and responsiveness

All AI-generated content and code were reviewed, modified, and tested to meet academic requirements.

## Features
- AI-based conversational companion
- Calm and minimal user interface
- Mindfulness-focused interactions
- Supportive and user-friendly design
- Responsive layout for different devices

## Technologies Used
- React
- Vite
- Tailwind CSS
- JavaScript
- AI-assisted development (Lovable)

## How to Run the Project Locally
1. Clone the repository  
   `git clone <repository-url>`
2. Navigate to the project folder  
   `cd mindful-muse`
3. Install dependencies  
   `npm install`
4. Start the development server  
   `npm run dev`
